Code Manual

The Family Pack uses Doxygen to document the program design and code. The doxygen configuration file for the TFP project is docs/doxygen/tfp.doxy. The doxygen program, version 1.6.3 or better, should be installed and on the path. For Windows users there is a batch file docs/doxygen/runDoxygen.bat which can be run to create the documentation files.
