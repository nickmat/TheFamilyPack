System

System table Core Data
[id]Meaning[val]Use
1Current User1The ID of the current User (ie U1)

The System table is used for storing settings related to the overal database. These values are handled directly by the program and whilst the settings can be edited, the user cannot add new entries. These entries form part of the Core Data. Currently, only one entry has been defined.

Current User

The ID of the current user and researcher. Many system settings are stored per user, and so this will also set other system wide enties. Eventually, this User/Reseacher will be be associated with any new data entries that are made.
