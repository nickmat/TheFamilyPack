Development Route Map

The Family Pack is a long term project but the speed of development will depend on many things. The following sets out the current intended order of development and anticipated milestones. Any dates are, however, deliberately missing.

V0.3.0.1 pre-alpha

This will be the first public version.

V0.3.0.x alpha

These will be the public alpha versions.

V0.3.1 beta

This will be the first public beta version.

From this point the versioning sytem will use a Major, Minor, Revision system where an even Minor number indicates a stable release with only (mostly?) bug fixes applied, whilst odd Minor numbers are used for developing new features and are therefore less stable.

V0.3.x beta

Beta versions developing a working User Interface.

V0.4.0 beta

First usable beta. (From this point, Databases will be guaranteed to be maintained across upgrades).

V0.4.x beta

Bug fixes to V0.4.0.

V0.5.x beta

Further development.

V1.0.0

First released stable version.

22nd March 2012
